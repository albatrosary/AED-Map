# Applications using the AED open data

## Installation

```bash
git clone git@github.com:albatrosary/AED-Map.git
cd AED-Map
npm i
```

## Usage

```bash
npm start
```

## Note

### Change API Key

Please use your API key

```html
<script async defer src="https://maps.googleapis.com/maps/api/js?key=[API Key]"></script>
```
